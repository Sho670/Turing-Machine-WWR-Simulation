Pattern Matching Using Nondeterministic Finite Automata (NFA)

Pattern matching is the process of checking whether an input string contains a specific sequence of characters (called a pattern). In automata theory, we can design Nondeterministic Finite Automata (NFA) that can detect whether a substring occurs anywhere inside a larger string.

NFAs allow multiple possible transitions from the same state using the same input symbol. This makes them ideal for pattern matching because the NFA can “guess” where the pattern might start. If any path leads to an accepting state, the whole input is accepted.

The NFA below checks for the occurrence of either substring "abc" or "ac" inside the input string over the alphabet {a, b, c}. As soon as one of these patterns is detected, the NFA enters an accepting state and remains accepting.
