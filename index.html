About the Turing Machine for L = { w wR | w ∈ {0,1}* }

This Turing Machine accepts all strings where the second half is the reverse of the first half. This is the classic wwR language, meaning a string followed immediately by its mirror image.

Example accepted strings: 00, 0110, 10100101 Example rejected strings: 01, 0101, 1100

How the Machine Works

  • The machine starts by reading the leftmost symbol.
  • It marks the first symbol (0 → B or 1 → B depending on path).
  • It then scans toward the right end seeking the matching final symbol.
  • If found, it erases (makes it B), returns left, and repeats.
  • If all symbols are matched correctly, it moves to the ACCEPTED state.
  • If a mismatch occurs, the machine halts without reaching accept.
